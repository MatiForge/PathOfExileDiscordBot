This bot is a comprehensive tool for the Path of Exile community, blending economic tracking with interactive entertainment. Its core functionality allows players to log their gains and losses using !win, !lost, and !stats, providing a clear overview of their balance in terms of Divine Orbs and Mirrors. Additionally, the bot features a built-in gambling simulator (!gambling) for users to test their luck.

For utility, the bot integrates with the poe.ninja API to provide live market data for key currencies (!currency) and top-priced Divination Cards (!cards). It also manages player builds via !setbuild and !builds, and features an interactive menu for browsing vendor recipes (!vendor). With a local web server for data accessibility and administrative tools like !clear, this bot serves as an all-in-one companion for optimizing gameplay and fostering server engagement.
